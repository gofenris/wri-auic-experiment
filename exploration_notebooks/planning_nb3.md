Notebook 3: Workshop ideas × data assessment

This notebook takes the 10 ideas from the ideation workshop and
systematically evaluates what data is available in the API to support each
one, what's missing, and what would need to come from elsewhere.

Objective: 
* Produce a structured gap analysis — for each idea, what data exists, what's absent, and what the implications are for feasibility.
* the assessment is a structured comparison of what the ideas require against what the API actually contains

Input: Live API data from Notebook 1 (or cached), plus the workshop synthesis CSV as a reference for idea descriptions and input requirements.
* Same API endpoints as Notebook 1 — either fetched live or from a cached response saved by Notebook 1
* The workshop synthesis file containing the 10 idea descriptions, their stated input requirements, workstream assignments, and feasibility notes
    * `ideation_workshop_synthesis_temp_20260601.csv`

Credentials: None required.
